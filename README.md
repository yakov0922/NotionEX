# NotionEX

**NotionEX** 是一款极简且强大的 Chrome 浏览器扩展，专为 Notion 用户设计。它能帮助你一键将当前网页内容保存到指定的 Notion 数据库中，支持多工作空间管理，让你在收集信息时保持高效与条理。

## 🎯 核心功能

*   **一键保存**：快速将当前网页标题及链接保存到 Notion，支持自动抓取网页图标。
*   **多工作空间支持**：这对于拥有多个 Notion 账号（如个人版与公司版）的用户非常实用。你可以配置多个 Notion Integration Token，并在保存时灵活切换。
*   **灵活的数据库选择**：自动加载当前工作空间下的所有数据库，支持快速搜索和选择目标数据库。
*   **智能字段匹配**：自动识别数据库的 `Title` 属性，无论你将其命名为“Name”、“Title”还是其他名称，插件都能正确写入。
*   **极简设计**：采用现代化的极简 UI 风格，无广告、无多余干扰，专注于核心功能。
*   **安全隐私**：你的 Notion Token 仅保存在本地浏览器存储（Chrome Storage）中，不会上传至任何第三方服务器。

## 🚀 安装指南

### 方法一：加载已解压的扩展程序（开发版）

1.  克隆或下载本项目代码到本地。
2.  在项目根目录下运行安装依赖并构建：
    ```bash
    npm install
    npm run build
    ```
3.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
4.  开启右上角的 **"开发者模式"**。
5.  点击左上角的 **"加载已解压的扩展程序"**。
6.  选择项目目录下的 `dist` 文件夹即可。

## 🛠 使用说明

1.  **配置 Token**：
    *   点击浏览器插件栏的 NotionEX 图标，打开扩展。
    *   点击右上角的设置图标，进入工作空间管理界面。
    *   添加你的 Notion Integration Token（需在 [Notion Developers](https://www.notion.so/my-integrations) 申请，并确保该 Integration 已通过 "Add connections" 邀请到目标页面或数据库）。
    *   你可以为不同的 Token 设置易记的别名（如 "个人笔记"、"工作文档"）。

2.  **保存网页**：
    *   在任意网页点击插件图标。
    *   选择要保存到的工作空间。
    *   从下拉列表中选择目标数据库。
    *   （可选）编辑标题。
    *   点击 **"保存到 Notion"** 按钮即可。

## 📦 技术栈

*   **React** + **TypeScript**：构建稳健的前端交互。
*   **Vite**：极速的开发与构建工具。
*   **CRXJS**：专为 Chrome 扩展开发的 Vite 插件，优化开发体验。
*   **Notion API**：通过官方 API 与 Notion 数据交互。
